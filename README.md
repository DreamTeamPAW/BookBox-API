# BookBox-API

BookBox application backend

## How to run
```bash
  npm ci
```

create ```.env``` file and fill credentials
```bash
  DB_USER=  
  DB_PASSWORD=
  JWT_SECRET=
```

Run API (swagger created automatically) with
```bash
npm run dev
```

Swagger documentation url:
```bash 
{url}:{port}/api-docs
```
Example: ```http://localhost:3000/api-docs```
